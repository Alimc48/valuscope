# 💰 ValuScope

> AI-powered startup valuation platform — DCF, Comparable Company, Cap Table, Exit Scenarios

**🌐 Live Demo:** https://ghostwhistle.github.io/valuscope/

---

## 💡 Bu Proje Nedir?

ValuScope, girişimcilerin **"Şirketim ne kadar eder?"** sorusunu profesyonel finansal yöntemlerle yanıtlamasına yardımcı olan tek dosyalık bir web uygulamasıdır.

İşletme okurken farkettim ki, startup değerleme bir "kara kutu" gibi algılanıyor. Halbuki DCF, Comparable Company, Stage-Based gibi yöntemler çok güçlü ama hesaplaması zaman alıyor. Bu araç, **5 dakikada profesyonel bir değerleme raporu** sunuyor.

---

## ✨ Öne Çıkan Özellikler

### 📊 4 Farklı Değerleme Yöntemi
- **DCF (Discounted Cash Flow)** — WACC + Terminal Value ile gelecek nakit akışları
- **Comparable Company** — 13 sektör için çarpan analizi
- **Revenue Multiple** — Hızlı gelir bazlı değerleme
- **Stage-Based** — Pre-Seed'den Growth'a tipik değerleme aralıkları

### 💼 Cap Table & Yatırım Analizi
- **Cap Table Simülasyonu** — Mevcut ve yeni yatırım sonrası ortaklık yapısı
- **Dilution Analizi** — Kurucu seyreltme hesaplaması
- **Yatırımcı Getiri** — ROI, IRR, Çarpan, Geri ödeme süresi
- **Funding Simülasyonu** — Pre-Seed → Series C tüm yatırım turları

### 🎯 İleri Seviye Analiz
- **Sensitivity Analizi** — 4 değişkenli interaktif kaydırıcılarla "what-if" senaryoları
- **DCF 5 Yıllık Projeksiyon** — Görsel nakit akışı tahmini
- **Exit Senaryoları** — IPO / Acquisition / Secondary Sale
- **Sektör Benchmark** — Gelir çarpanı, EV/EBITDA, büyüme, marj karşılaştırması

### 🤖 Yapay Zeka Destekli
- **AI Değerleme Raporu** — Verilere özel kapsamlı strateji
- **AI Q&A** — Değerleme hakkında sorular sorabilirsin

### 🌍 Diğer
- 🇹🇷🇬🇧 TR/EN bilingual desteği
- 💱 12 farklı para birimi
- 🌓 Light/Dark tema
- 📊 PDF export
- 💾 LocalStorage ile geçmiş analiz kaydı
